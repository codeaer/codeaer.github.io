为什么会有这个博客

by 阿秀
2023-08-17
约 555 字 - 预计阅读 2 分钟
博客之路

欢迎来到码农小站，欢迎阅读这篇博客，这是小站的第一篇博客，希望你会喜欢。

不得不承认，现在是短视频时代，坚持写博客的人越来越少了，以前喜欢的订阅的博客也好久没有更新了。

都 2023 年了，谁还会想着写博客呢？是我，我又开始写博客了，为什么是“又”呢？

其实之前折腾过好几次博客，最开始写博客是在 2018 年，当时看了阮一峰大佬的博客，觉得自己也应该写博客。

后来想通过写博客赚钱，在 CSDN、简书、知乎等平台写过博客，在百家号、头条号等平台写过博客。

还从零开发过几次博客程序，购买域名，备案域名，购买服务器，开发一大堆功能，一顿操作下来，没有写几篇博客文章。

折腾几次最后都没有坚持下来，这次能坚持下来吗？答案是不一定，所以在此立 flag，到明年的今天写下 42 篇博客。

搭建这个博客主要是为了记录和分享，记录自己学习开发中遇到的问题，下次再遇到就不需要百度了。分享是为了帮助遇到同样问题的人，毕竟愿意写博客的人少了，能搜索到的解决方案就更少。

我是很喜欢逛别人的博客，尤其是工作摸鱼的时候，看看别人的经历，再照照镜子，或许会有这人的经历跟我好像的感叹。写博客的人少了，我就想为同样喜欢逛博客的人多增加一个选择。

这个博客是使用 Hugo + Github Pages + Github Action 免费搭建的，这也避免了因为缺钱支付不起服务器费用而选择放弃博客，在 Github Pages 不改变的情况，这个博客会一直存在。
